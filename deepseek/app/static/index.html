Widget Demonstrations

This page demonstrates the two AI assistant widgets integrated with FastAPI. You can embed these into your own website using the code snippets below.

User Assistant Widget

This widget uses the regular RAG chain to help employees navigate changes using the ADKAR model.

Note: This widget is accessible to all users without authentication.

Integration Instructions

To integrate these widgets into your website:

  1. Deploy the FastAPI backend on your server or cloud provider
  2. Update the API_URL in the widget HTML files to point to your deployed API
  3. Host the widget HTML files on your web server
  4. Embed the widgets in your website using the iframe code shown above
  5. Customize the styling to match your website's design

Security Note: In production, make sure to:

  • Use HTTPS for all communications
  • Set proper CORS policies on your API server
